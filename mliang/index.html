Firefox OS Projects

Firefox OS - Smart TV Prototype

Firefox OS Smart TV is one of our main products in the Taipei office. While we were developing the project, we wanted to test out some new features so I made a high-fidelity prototype implemented with the features we wanted to test. We've used this prototype to let multiple participants to navigate through and try out new functions using actual remote control (This web version prototype can be control by keyboard.)

Firefox OS - Add-On Prototype

This Add-on prototype is one of our early concepts about adding a universal button on top of our Firefox OS interface. It acts as a floating menu button with the most used function in it, such as "Home", "Back" and "Task Manager". I used Firefox Web IDE tools to develop this add-on using basic Javascript and CSS.

Connected Devices Projects

Connected Devices - Project SensorWeb Website

Project SensorWeb is an open data platform for IoT. Our platform allows you to create REST APIs through a web interface to easily share data from IoT devices without coding. SensorWeb empowers developers to collectively solve problems they care about. This is a portal website where we provide useful information about our services.

http://sensorweb.io/

Table of Contents Sketch Plugin

This is a Sketch Plugin I created upon a request from our UX team. As a memember of the Taipei UX team we usually have to create different specs using Sketch, and we have to manually add a "Table of Contents" page to list out the contents. For this plugin you just have to press the shortcut and it will automatically generate a Table of Contents page for you. (There are some rules to follow when you create your spec)
